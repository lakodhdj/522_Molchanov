# Система управления пользователями и платежами

WPF-приложение для учёта финансовых операций, администрирования пользователей и визуализации данных с поддержкой экспорта отчётов.

---

## Описание проекта

Данное настольное приложение предназначено для ведения учёта платежей и управления пользователями с разграничением прав доступа. В зависимости от роли — администратора или обычного пользователя — предоставляется различный функционал.

Проект сочетает удобный интерфейс, безопасность данных и гибкость настройки, обеспечивая эффективную работу с финансовой информацией и её анализом.

---

## Основные возможности

### Безопасность и авторизация

* Регистрация и вход с проверкой ролей
* Хеширование паролей (SHA1)
* Валидация вводимых данных
* Капча при множественных неудачных попытках входа
* Смена пароля и проверка его сложности

### Управление пользователями

* Просмотр, добавление, редактирование и удаление пользователей
* Разделение ролей (администратор/пользователь)
* Фильтрация и сортировка списка
* Поддержка фотографий профиля

### Работа с платежами и категориями

* Учёт финансовых операций по категориям
* CRUD-операции для платежей и категорий
* Группировка и поиск по различным параметрам

### Визуализация и отчётность

* Отображение статистики расходов в виде диаграмм
* Поддержка нескольких типов графиков (круговые, столбчатые и др.)
* Экспорт данных в **Excel**, **Word** и **PDF**
* Формирование отчётов с итоговыми суммами и форматированием

## Архитектура приложения

### Основные модули

* **AuthPage** — авторизация
* **RegPage** — регистрация
* **AdminPage / UserPage** — главные панели ролей
* **DiagrammPage** — визуализация данных
* **UsersTabPage**, **CategoryTabPage**, **PaymentTabPage** — табличные представления
* **Add/Edit Pages** — формы добавления и изменения данных

### Используемые технологии

* **WPF** — реализация интерфейса
* **Entity Framework** — доступ к базе данных
* **SQL Server** — хранение данных
* **Office Interop** — экспорт отчётов
* **Windows Forms Integration** — визуализация диаграмм

## Установка и запуск

### Требования

* .NET Framework 4.7.2+
* Microsoft SQL Server 
* Microsoft Office для экспорта

### Этапы запуска

1. Клонировать репозиторий с GitHub
2. Восстановить зависимости NuGet
3. Настроить строку подключения к базе данных
4. Запустить приложение через Visual Studio
